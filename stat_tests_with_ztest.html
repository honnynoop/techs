t-검정 (t-test)

모수적 검정
📌 단일표본 t-검정 (One-sample t-test)

원리 (Principle)

모집단 평균이 특정 값 μ₀과 같은지 검정. 표본 평균과 모집단 가설값의 차이를 표준오차로 나누어 t통계량 계산.

t = (x̄ - μ₀) / (s / √n)
// df = n - 1
// x̄: 표본 평균, μ₀: 가설 평균
// s: 표본 표준편차, n: 표본 크기

가정 (Assumptions)

  • 데이터가 정규분포를 따름
  • 표본이 모집단에서 독립적으로 추출
  • 연속형 데이터
  • n ≥ 30 이면 CLT에 의해 완화 가능

언제 사용?

🏭 품질 관리제품 평균 무게가 규격(500g)과 다른지
📚 교육학급 평균 점수가 전국 평균 70점과 다른지
🏥 의료혈압 평균이 정상범위 기준값과 다른지
📊 수치 예제 — 커피 용량 품질 검사 α = 0.05, 양측검정

어떤 커피 제조사는 1캔에 μ₀ = 350mL가 들어있다고 주장한다. 10캔을 샘플링해 확인하였다.

통계적으로 커피 용량은 350mL와 유의한 차이가 없다.
📌 독립표본 t-검정 (Independent samples t-test)

원리

서로 독립된 두 집단의 모집단 평균이 같은지 검정. Levene 등분산 검정 후 등분산/이분산 공식 선택.

// 등분산 가정 시 (pooled) t = (x̄₁-x̄₂) / Sp·√(1/n₁+1/n₂)
Sp² = [(n₁-1)s₁²+(n₂-1)s₂²]/(n₁+n₂-2)
// df = n₁+n₂-2
// 이분산 시 Welch's t-test 사용
  • 두 그룹이 서로 독립
  • 각 집단 내 정규분포
  • 등분산성 (Levene 검정으로 확인)

언제 사용?

💊 임상시험치료군 vs 대조군 혈압 차이
🎓 교육A반 vs B반 시험 점수 비교
🏭 생산기계A vs 기계B 불량률 비교
📊 수치 예제 — 두 교수법의 시험 점수 비교 α = 0.05, 양측

📌 대응표본 t-검정 (Paired samples t-test)

원리

동일 대상에서 두 번 측정(사전-사후, 왼손-오른손)한 차이(d)를 분석. 개인차를 제거해 검정력 향상.

d_i = x₁ᵢ - x₂ᵢ // 쌍별 차이
t = d̄ / (s_d / √n)
// d̄: 차이의 평균
// s_d: 차이의 표준편차, df=n-1
  • 쌍이 이루어진 관측값
  • 차이 d가 정규분포
  • 각 쌍은 독립

언제 사용?

💊 효과 검증다이어트 프로그램 전후 체중
📚 교육강의 전후 이해도 점수
🏃 스포츠훈련 전후 기록 변화
📊 수치 예제 — 다이어트 프로그램 전후 체중 α = 0.05, 단측(감소)

d̄ = (4+3+4+3+5+4+6)/7 = 29/7 = 4.143

ANOVA (분산분석)

모수적 검정
📌 일원배치 분산분석 (One-way ANOVA)

원리

3개 이상 집단 평균 비교. 집단 간 분산(처리 효과)을 집단 내 분산(오차)으로 나눈 F통계량 사용.

SS_총 = Σ(x_ij - x̄_총)² SS_집단간 = Σ nⱼ(x̄ⱼ - x̄_총)² SS_집단내 = SS_총 - SS_집단간
F = MS_집단간 / MS_집단내 = (SS_B/k-1) / (SS_W/N-k)
  • 각 집단 정규분포
  • 집단 간 분산 동일 (등분산)
  • 관측값 독립

언제 사용?

💊 임상3가지 약물 치료 효과 비교
🌾 농업4종 비료별 수확량 차이
🏭 제조3개 공장 불량률 차이
⚠️ 주의: ANOVA는 어딘가 차이가 있다는 것만 알려줌. 어디서 차이나는지는 사후검정 필요.

📌 사후검정 (Post-hoc Tests)

왜 필요한가?

ANOVA가 유의하면 어떤 집단쌍이 다른지 알아야 함. 단순 t-test 반복 시 1종 오류 누적 → 가족별 오류율 보정 필요.

Tukey HSD — 모든 쌍 비교, 가장 일반적
HSD = q × √(MS_W/n)
Bonferroni — 보수적, α/비교수로 보정
Scheffé — 가장 보수적, 복합 비교

위 예제 Tukey HSD 적용

비료 A vs B: |20.5-29.0| = 8.5 비료 A vs C: |20.5-19.75| = 0.75 비료 B vs C: |29.0-19.75| = 9.25 HSD = q(3,9,0.05) × √(2.639/4) = 3.95 × 0.8121 = 3.208 A vs B: 8.5 > 3.208 → 유의 A vs C: 0.75 < 3.208 → 비유의 B vs C: 9.25 > 3.208 → 유의
결론: 비료 B가 A, C보다 유의하게 높음. A와 C는 차이 없음.

카이제곱 검정 (χ²)

범주형 데이터
📌 카이제곱 독립성 검정 (χ² Test of Independence)

원리

두 범주형 변수가 서로 독립인지 검정. 관측빈도와 기대빈도의 차이로 χ² 계산.

χ² = Σ (O_ij - E_ij)² / E_ij
E_ij = (행합_i × 열합_j) / N
// df = (행수-1)×(열수-1)
  • 범주형 데이터
  • 기대빈도 E ≥ 5 (각 셀)
  • 관측값 독립
  • E < 5 셀 있으면 Fisher's Exact Test

언제 사용?

📣 마케팅성별과 상품 선호도 관련성
🏥 역학흡연 여부와 폐암 발병 관련성
📊 여론지역별 정책 지지 여부
📊 수치 예제 — 성별과 운동 선호 종목 관련성 α = 0.05

남녀 각 100명에게 선호 운동(축구/농구/수영) 조사

E(남,수영) = 100×80/200 = 40.0  |  E(여,축구) = 32.5  |  E(여,농구) = 27.5  |  E(여,수영) = 40.0

📌 카이제곱 적합도 검정 (χ² Goodness of Fit)

원리

관측된 빈도 분포가 이론적(기대) 분포와 일치하는지 검정. 1개 범주형 변수.

χ² = Σ (O_i - E_i)² / E_i
// df = 범주수 - 1
// E_i = N × p_i (이론 확률)

언제 사용?

🎲 확률주사위가 공정한지 (1~6 균등)
📊 시장제품별 시장점유율이 기대와 같은지
📊 수치 예제 — 주사위 공정성 검정α = 0.05

✅ 결론: χ²(5) = 3.50 < 11.07, p ≈ 0.623 → H₀ 채택
주사위가 공정하지 않다는 증거 없음.

비모수 검정 (Non-parametric)

정규성 불필요
📌 Mann-Whitney U 검정 독립표본 t-test의 비모수 대안

원리

두 집단 값을 합쳐 순위 매긴 후, 각 집단의 순위합을 비교. 중위수 차이 검정.

U₁ = n₁n₂ + n₁(n₁+1)/2 - R₁ U₂ = n₁n₂ + n₂(n₂+1)/2 - R₂ // R₁,R₂: 각 집단 순위합 U = min(U₁, U₂) // 최솟값 사용

언제 사용?

📈 소규모n이 작고 정규성 가정 불확실
🏆 순위형리커트 척도, 만족도 점수
⚡ 이상치극단값이 있는 데이터
📊 수치 예제 — 두 교육법 만족도 비교α=0.05, n₁=n₂=6
방법 A354253
방법 B768597
1
합쳐서 순위 배정
정렬: 2(A),3(A),3(A),4(A),5(A),5(A),5(B),6(B),7(B),7(B),8(B),9(B)
순위: 1, 2.5, 2.5, 4, 6, 6, 6, 8, 9.5, 9.5, 11, 12
2
집단별 순위합
R_A = 1+2.5+2.5+4+6+6 = 22.0
R_B = 6+8+9.5+9.5+11+12 = 56.0
3
U 통계량
U_A = 6×6 + 6×7/2 - 22 = 36+21-22 = 35
U_B = 6×6 + 6×7/2 - 56 = 36+21-56 = 1
U = min(35,1) = U = 1
4
판정
U_critical(6,6,0.05 양측) = 5. U=1 ≤ 5 → 유의
🔴 결론: U=1, p ≈ 0.004 → H₀ 기각. 방법 B의 만족도가 유의하게 높음.
📌 Wilcoxon 부호순위 검정 대응표본 t-test의 비모수 대안

원리

쌍별 차이 |d|를 순위 매긴 후, 양의 부호(+)와 음의 부호(-) 순위합 비교.

W+ = 양의 차이 순위합 W- = 음의 차이 순위합 T = min(W+, W-) // T가 임계값보다 작으면 기각

📌 Kruskal-Wallis 검정 One-way ANOVA의 비모수 대안

원리

3개 이상 집단의 순위합 비교. 전체 순위 배정 후 집단별 평균 순위 차이로 H통계량 계산.

H = [12/(N(N+1))] × Σ(Rⱼ²/nⱼ) - 3(N+1)
// N: 전체 n, Rⱼ: j집단 순위합 // df = k-1, χ²분포 근사

예제 — 3지역 집값 비교 (단위: 억)

A지역3.24.12.83.9
B지역5.16.34.85.9
C지역2.13.01.82.5
순위 배정 후: R_A=22, R_B=38, R_C=6
H = [12/(12×13)] × [(22²/4)+(38²/4)+(6²/4)] - 3×13
= 0.07692 × [121+361+9] - 39 = 37.77 - 39 = H = 10.38
df=2, χ²_critical(2,0.05)=5.99 → p ≈ 0.006 → H₀ 기각

상관분석 & 회귀분석

📌 Pearson 상관계수 (r)

원리

두 연속형 변수 간 선형 관계 강도·방향. -1 ≤ r ≤ 1. 두 변수 모두 정규분포 가정.

r = Σ[(xᵢ-x̄)(yᵢ-ȳ)] / √[Σ(xᵢ-x̄)²·Σ(yᵢ-ȳ)²]
// 유의성 검정: t = r√(n-2)/√(1-r²) // df = n-2

📌 단순 선형 회귀 (Simple Linear Regression)

원리

X → Y의 인과적 관계를 직선으로 모델링. 최소제곱법으로 회귀계수 추정.

Z 검정 (Z-test)

모수적 검정 n≥30 or σ 기지
📌 Z 검정의 핵심 원리

Z 검정은 모집단 표준편차(σ)가 알려져 있거나 표본 크기 n이 충분히 클 때(n≥30) 사용한다. 중심극한정리(CLT)에 의해 표본 평균이 정규분포를 따르므로, t 분포 대신 표준정규분포 N(0,1)을 직접 사용한다.

Z = (관측 통계량 - 귀무가설 기준값) / 표준오차
// Z ~ N(0,1) 표준정규분포 // 양측검정: |Z| > 1.960 (α=0.05) // 우측단측: Z > 1.645 (α=0.05) // 좌측단측: Z < -1.645 (α=0.05)
📋 Z 검정의 3가지 종류

① 단일표본 Z 검정

하나의 표본 평균이 모평균 μ₀와 같은지
조건: σ 알려짐 or n≥30
통계량: Z = (x̄-μ₀)/(σ/√n)

② 이표본 Z 검정

두 집단 평균 차이 검정
조건: 두 σ₁, σ₂ 모두 알려짐
통계량: Z = (x̄₁-x̄₂)/SE

③ 비율 Z 검정

표본 비율이 모비율 p₀와 같은지
조건: np₀≥5, n(1-p₀)≥5
통계량: Z = (p̂-p₀)/SE
📊 임계값 & p-value 빠른 참조표
검정 방향α=0.10α=0.05α=0.01α=0.001
양측 |Z| >1.6451.9602.5763.291
우측단측 Z >1.2821.6452.3263.090
좌측단측 Z <-1.282-1.645-2.326-3.090
선택 가이드: σ를 아는가? → Z 검정 사용. 모르면 n≥30인가? → Z 또는 t 모두 가능. n<30이면 → 반드시 t 검정 사용.
📌 단일표본 Z 검정 (One-sample Z-test)

원리 (Principle)

하나의 표본 평균이 주어진 모평균 μ₀와 유의하게 다른지 검정. σ가 알려져 있을 때 t 검정 대신 사용. 표준오차 = σ/√n.

Z = (x̄ - μ₀) / (σ / √n)
// x̄: 표본 평균 // μ₀: 귀무가설의 모집단 평균 // σ: 모표준편차 (알려진 값) // n≥30이면 s로 σ 대체 가능

가정

  • 정규분포 or n≥30 (CLT)
  • 모표준편차 σ 알려짐
  • 독립 표본
  • 연속형 데이터
  • σ 모를 경우 t 검정 사용

언제 사용?

🏭 품질관리역사적 σ 알려진 제조 공정 규격 검사
🧠 심리검사IQ 표준화 검사 (σ=15 고정)
📊 대규모 조사n≥30 이상 여론조사 평균 비교
📊 수치 예제 — 배터리 수명 검사 α=0.05, 양측검정

제조사는 배터리 평균 수명이 μ₀=500시간이라 주장. 과거 이력에서 σ=40시간 알려짐. n=36개 샘플 측정 결과 x̄=490.8시간.

1
가설 설정
H₀: μ = 500 (주장 맞음) / H₁: μ ≠ 500 (다름) — 양측
2
표준오차 계산
SE = σ/√n = 40/√36 = 40/6 = 6.667시간
3
Z 통계량
Z = (490.8 - 500) / 6.667 = -9.2 / 6.667 = Z = -1.380
4
판정
α=0.05 양측 → Z_critical = ±1.960
|Z| = 1.380 < 1.960 → 기각역 밖
p-value = 2×P(Z < -1.38) = 2×0.0838 = 0.168
✅ 결론: Z = -1.380, p = 0.168 > 0.05 → H₀ 채택
표본 평균 490.8시간은 주장값 500시간과 통계적으로 유의한 차이 없음.
📌 표본 크기 효과 비교:
n=36 → SE=6.667 → Z=-1.38 → p=0.168 → H₀ 채택
n=100 → SE=4.000 → Z=-2.30 → p=0.021 → H₀ 기각!
→ n이 클수록 동일한 차이도 유의해짐 (검정력 증가)
📌 이표본 Z 검정 (Two-sample Z-test)

원리

두 독립 집단의 모평균 차이를 검정. 두 집단의 σ₁, σ₂가 모두 알려진 경우 사용.

Z = (x̄₁ - x̄₂) / √(σ₁²/n₁ + σ₂²/n₂)
// 두 σ 모두 알려져야 함 // n₁,n₂ 모두 ≥30이면 s₁,s₂로 대체 // H₀: μ₁ - μ₂ = 0
  • 두 집단 독립
  • 각 집단 정규분포 or n≥30
  • σ₁, σ₂ 모두 알려짐

언제 사용?

🏭 제조두 공장의 제품 규격 차이 (σ 이력 있음)
🏥 임상대규모 시험 두 치료군 비교
📈 금융두 종목의 평균 수익률 비교
📊 수치 예제 — 두 공장 부품 길이 비교 α=0.05, 양측
공장nx̄ (mm)σ (알려짐)
A5050.43
B6049.14
1
가설
H₀: μ_A = μ_B / H₁: μ_A ≠ μ_B — 양측검정
2
표준오차
SE = √(σ₁²/n₁ + σ₂²/n₂) = √(9/50 + 16/60) = √(0.180 + 0.267) = √0.447 = 0.6685
3
Z 통계량
Z = (50.4 - 49.1) / 0.6685 = 1.3 / 0.6685 = Z = 1.945
4
판정
|Z|=1.945 vs Z_critical=1.960 → 간신히 기각 못함
p-value = 2×(1-Φ(1.945)) = 2×0.0259 = 0.0518 > 0.05
✅ 결론: Z=1.945, p=0.052 > 0.05 → H₀ 채택 (아주 근소하게)
두 공장 부품 길이 차이(1.3mm)는 5% 유의수준에서 통계적으로 유의하지 않음.
* α=0.10 기준이었다면 p=0.052 < 0.10 → H₀ 기각됨. 유의수준 선택의 중요성.
📌 비율 Z 검정 (Proportion Z-test)

단일 비율 Z

Z = (p̂ - p₀) / √(p₀(1-p₀)/n)
// p̂ = x/n : 표본 비율 // p₀ : 귀무가설의 모비율 // 적용 조건: np₀≥5, n(1-p₀)≥5

두 비율 Z

Z = (p̂₁-p̂₂) / √[p̂(1-p̂)(1/n₁+1/n₂)]
// p̂ = (x₁+x₂)/(n₁+n₂) 합동비율

언제 사용?

🏭 품질불량률이 허용 기준 5%보다 높은지
📣 마케팅A/B 테스트 클릭률 비교
🗳 여론지지율이 50%를 넘는지
🏥 의학치료군 vs 대조군 완치율 비교
핵심: 비율 검정은 Z를 가장 많이 쓰는 분야. t 검정보다 Z 검정이 표준.
📊 예제 1 — 단일 비율: 불량률 기준 초과 여부 α=0.05, 우측단측

공장 불량률 기준 p₀=5%. n=200개 검사 → 불량 x=16개 발견. 불량률이 증가했는가?

1
가설
H₀: p = 0.05 / H₁: p > 0.05 — 우측단측
2
표본 비율 & 조건 확인
p̂ = 16/200 = 0.08 (8%)
np₀=200×0.05=10≥5 ✓  |  n(1-p₀)=190≥5 ✓
3
Z 계산
SE = √(p₀(1-p₀)/n) = √(0.05×0.95/200) = √0.0002375 = 0.01541
Z = (0.08 - 0.05) / 0.01541 = 0.03 / 0.01541 = Z = 1.947
4
판정
우측단측 Z_critical = 1.645
Z = 1.947 > 1.645 → 기각역 내부
p-value = P(Z > 1.947) = 0.0258
🔴 결론: Z=1.947, p=0.026 < 0.05 → H₀ 기각
불량률이 기준 5%에서 통계적으로 유의하게 증가했다 (관측 불량률 8%).
